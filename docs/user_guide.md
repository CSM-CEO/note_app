# Руководство пользователя: Note App

Приложение "Заметки с тегами" предназначено для быстрого и удобного структурирования текстовой информации с помощью системы тегов.

## Как пользоваться приложением:

1. **Создание заметки:**
   * В левой панели заполните поле "Заголовок" и "Текст заметки".
   * В поле "Теги" введите ключевые слова через запятую (например: `учеба, важно, python`).
   * Нажмите кнопку **"Создать запись"**.

2. **Просмотр и чтение заметок:**
   * Все заметки отображаются в центральной колонке в хронологическом порядке (сверху самые свежие).
   * Под каждой заметкой выводятся привязанные к ней теги в виде кликабельных кнопок.

3. **Фильтрация по тегам:**
   * Чтобы увидеть только заметки на определенную тему, нажмите на интересующий вас тег под любой заметкой или в правом блоке "Все теги".
   * Чтобы вернуться к полному списку, нажмите кнопку **"Сбросить фильтр"** в появившейся информационной панели.

4. **Полнотекстовый поиск:**
   * В верхней навигационной панели введите любое слово или фразу в поисковую строку и нажмите **"Найти"**. Приложение отфильтрует записи по совпадению в заголовке или тексте.

5. **Удаление записей:**
   * Для удаления заметки нажмите на иконку "корзины" в правом верхнем углу карточки заметки и подтвердите действие в окне браузера.
